What?

Thriftify is a simple web service which generates a Thrift bindings from text, uploaded files or URLs

Why?

Most people don't need to compile and install the Thrift compiler. What they do need are the Thrift runtime libraries per their chosen bindings and the bindings. We simply wanted to save that.

How?

It started with a set of tweets from @scode and @rantav. At some point @erans saw a retweet of @scode by @rantav and thought it would be a nice side project.
