API

A simple one-liner:

    $ curl "http://thriftify.org/api/v1/generate?gen=java" -F "file1=@file.thrift" > java.zip

A complete example:

    # Create a simple thrift file
    $ echo struct A {} > file.thrift
    # Generate the thrift bindings using thriftify
    $ curl "http://thriftify.org/api/v1/generate?gen=java" -F "file1=@file.thrift" > java.zip
    # Validate, read the code
    $ unzip -c java.zip
